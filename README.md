# Table Data Display - Mulah Internship Assessment

A web application that displays CSV data in two formatted tables as per the internship assessment requirements.

## Overview

This application displays:

1. **Table 1**: Original CSV data with Index # and Value columns
2. **Table 2**: Calculated values from the original data:
   - **Alpha**: A5 + A20 (2 + 28 = 30)
   - **Beta**: A15 / A7 (80 / 5 = 16)
   - **Charlie**: A13 * A12 (90 * 3 = 270)

## Features

- Responsive web design
- Real-time data processing
- Clean, professional table layout
- Error handling and loading states
- Mobile-friendly interface

## Technology Stack

- **Backend**: Node.js with Express.js
- **CSV Processing**: csv-parser library for reading CSV files
- **Frontend**: HTML5, CSS3, Vanilla JavaScript
- **Deployment**: Render
